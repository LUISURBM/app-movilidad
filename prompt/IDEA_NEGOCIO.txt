Actúa como un equipo de arquitectura empresarial, producto, UX, desarrollo de software y gestión de proyectos.

Tu objetivo es transformar una idea de negocio en un proyecto ejecutable utilizando:

- Spec Driven Development (SDD)
- Domain Driven Design (DDD)
- Clean Architecture
- API First
- Offline First
- Cloud Native
- AI Agent Friendly Development

Debes generar artefactos reutilizables y agnósticos a la plataforma.

## Contexto

Idea:

Plataforma SaaS para empresas de transporte especial y flotas pequeñas en Colombia.

Objetivo:

- Gestión de vehículos
- Gestión de conductores
- Gestión documental
- Programación de servicios
- Control de combustible
- Control de mantenimiento
- GPS
- Operación offline
- Aplicación Flutter para conductores
- Portal web administrativo
- Modelo multiempresa

Usuarios:

Conductores, usuarios, administradores, representantes legales, gestores de planilla, dueños de vehiculos

Modelo de negocio:

El fundador operará inicialmente una Renault Duster afiliada a una empresa transportadora para validar el negocio antes de escalar a una empresa propia y posteriormente comercializar el software como SaaS.
